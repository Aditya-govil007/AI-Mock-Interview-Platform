# 🤖 AI Mock Interview Platform

An AI-powered mock interview platform that simulates real interview scenarios and provides intelligent feedback to help users improve their communication and technical skills.

---

## 🚀 Features

- 🎤 **AI-based Mock Interviews** – Simulates real interview experience  
- 🤖 **Smart Feedback System** – Generates answers analysis using AI  
- 🧠 **Dynamic Question Generation** – Based on role & tech stack  
- 🎙️ **Voice Recording Support** – Record and analyze spoken answers  
- 📊 **Performance Evaluation** – Feedback on communication & content  
- 🔐 **Authentication System** – Secure login/signup  
- 🌐 **Modern UI** – Clean and responsive interface

## ⚙️ Installation & Setup

1. Clone the repository  
```bash
git clone https://github.com/Aditya-govil007/AI-Mock-Interview-Platform.git
```
2. Navigate to the project directory
``` bash
cd AI-Mock-Interview-Platform
```
3.Install dependencies
```bash
npm install
```
4.Add environment variables
  Create a .env.local file and add:
  ```bash
   GOOGLE_API_KEY=your_api_key
  ```
5.Run the development server
```bash
npm run dev
```
